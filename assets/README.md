# Assets do Jogo

Coloque aqui os assets do jogo Umamusume Pretty Derby que você quer usar na calculadora.

## Estrutura de Pastas

- `images/` - Imagens de personagens, ícones, etc.
  - `character.png` - Imagem do personagem (recomendado: 100x100px ou maior)
  
- `icons/` - Ícones do jogo
  - Ícones de estrelas, botões, etc.

## Formatos Suportados

- Imagens: PNG, JPG, JPEG, GIF, WebP
- Tamanho recomendado para character.png: 100x100px ou maior (será redimensionado automaticamente)

## Como Usar

1. Coloque a imagem do personagem em `assets/images/character.png`
2. A imagem será exibida automaticamente na interface
3. Se a imagem não for encontrada, um placeholder será exibido
